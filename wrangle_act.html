Gathering Data¶

The project gathers three pieces of data as described below:

  • Twitter archive file: This can be downloaded manually via twitter_archive_enhanced.csv and read into a pandas DataFrame.

  • The tweet image predictions: This is present in each tweet according to a neural network. It is hosted on Udacity's servers and should be downloaded programmatically using the Requests library and the following URL: tweet_image_predictions. The results is a table full of image predictions (the top three only) alongside each tweet ID, image URL, and the image number that corresponds to the most confident prediction (numbered 1 to 4 since tweets can have up to four images).

  • tweet's JSON data should be written to its own line. Then read line by line into a pandas DataFrame with (at minimum) tweet ID, retweet count, and favorite count.

Assessing Data¶

After gathering all three pieces of data, assess them visually and programmatically for quality and tidiness issues. Detect and document at least eight (8) quality issues and two (2) tidiness issues.

You need to use two types of assessment:

  • Visual assessment: each piece of gathered data is displayed in the Jupyter Notebook for visual assessment purposes. Once displayed, data can additionally be assessed in an external application (e.g. Excel, text editor).
  • Programmatic assessment: pandas' functions and/or methods are used to assess the data.

Quality¶

twitter_archive table¶

  1. timestamp dtype format.
  2. change rating_denominator values rated above 10.
  3. fill out-of-range values of rating_numerator.
  4. convert the source column to category type.
  5. drop columns with null values.

Image Prediction table¶

  1. select 'tweet_id', 'jpg_url', 'img_num'

Tweet json table¶

  1. select 'id', 'retweet_count', 'favorite_count' columns
  2. change column name id to tweet_id
  3. interpolate retweet_count and favorite_count with the mean.

Tidiness¶

  1. Remove the leading html tags from the source column
  2. Melt the doggo, floofer, pupper and puppo columns as dog_stages column.
  3. merge the three tables as one and drop all duplicated rows.

Conclusions:¶

To sum up everything that has been stated so far, there is a positive correlation of 0.79 between the favorite count and retweet count variables. Thus, favorite tweets are more likey to be retweeted. Mean numerator rating above 10 have significantly higher favorite and retweet counts. However, posts with more images does not necessarily lead to better engagement.

Limitations:¶

Some dogs have multiple stages and if these stages were to be melted, this would result in the dataset being duplicated.
